# laravel5.5 + react

# Requirements
* PHP >=7.0
* Laravel >= 5.5

# Installation

## 1. 
```
composer install
```

## 2.

```
cp .env.example .env

vim .env
```

## 3.

```
php artisan key:generate

```

## 4.

```
php artisan migrate
```

## 5.

```
npm install

npm run dev
```

# Usage
```
php artisan serve
```
